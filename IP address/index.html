Thought process, how did I solve the task?
  1. At first, it was just one blurry text for me with no meaning at all.
  2. I started looking for any patterns in the text itself, but nothing came up.
  3. Then I noticed that the command was talking about 10 lines of text, but it still didn't look like lines, just an infinite string of characters. So I started adding more columns in excel to see how long the text was but it still seemed endless.
  4. So I decided to copy everything to notepad to see what it looks like without the styles imposed by excel. In the end, 10 separate paragraphs appeared to me.
  5. After a while, I realized that there must be a decoding pattern for this. That is, a string that will check whether there is a hidden IP in a given fragment of text.
  6. The idea itself came from the fact that a few days ago in JavaSscript I was writing a form with a pattern to check the correctness of the email address entered in the input.
  7. Finally, I decided to write a function that, based on the array, will check each paragraph to see if there is a hidden IP. As you can see above, we have the result.
